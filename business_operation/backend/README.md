# 百惠康大药房业务运营后端系统

## 项目概述

百惠康大药房网上药店经营系统B端后端服务，为药店管理人员提供商品管理、订单处理、统计分析等功能。

## 技术栈

- **框架**: Spring Boot 2.7.18
- **ORM**: MyBatis Plus 3.5.3.1
- **数据库**: MySQL 8.0（共用C端数据库）
- **缓存**: Redis（数据库1）
- **认证**: JWT + BCrypt
- **Excel**: Apache POI

## 核心功能

### 1. 账号密码登录
- 管理员账号密码登录
- BCrypt密码加密
- JWT Token认证
- 修改密码

### 2. 商品管理子系统
- 商品增删改查
- 商品上下架
- 库存查看

### 3. 订单处理子系统
- 订单列表查询
- 发货处理
- 售后审核

### 4. 统计分析子系统
- 销售报表（日/周/月/年）
- 业绩统计
- 商品销售统计
- Excel导出

## 默认账号

- **账号**: admin
- **密码**: admin123

## 端口配置

- **端口**: 8081
- **路径**: /admin-api

## 启动命令

```bash
cd business_operation/backend
mvn spring-boot:run
```

访问：http://localhost:8081/admin-api
